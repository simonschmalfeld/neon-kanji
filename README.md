# Neon Kanji

A p5.js visual that displays random Japanese kanji in a neon Tokyo / manga aesthetic.

## Features

- 30 hand-picked kanji with kana, romaji, and English translation
- Multi-layer neon glow with animated pulse and occasional flicker
- Glitchy transition effect between kanji (auto every ~5s, or click to skip)
- Ambient background kanji drifting upward
- Perspective grid, scanlines, and floating particles
- 3 rotating color palettes (red/purple, orange/magenta, purple/red)
- No repeats for the last 10 kanji shown

## Usage

Open `index.html` directly in a browser — no build step or server required.

Click anywhere to trigger the next kanji immediately.

## Data source

Kanji readings and meanings sourced via [kanjiapi.dev](https://kanjiapi.dev).
